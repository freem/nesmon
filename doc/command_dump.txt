quick command list
* A                    assemble a single line of code
* C                    comparison
* CLS                  clear the screen
* D                    disassemble
* JMP                  jmp
* R                    show regs
* RESET                reset system
* X                    quit monitor


* run                  

unsure command list
* BTL                  run serial bootloader
* TPL                  load from tape (Family BASIC Keyboard only)
* TPS                  save to tape (Family BASIC Keyboard only)
* clearall             clear all RAM
